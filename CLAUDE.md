# Query-Hub — Claude Code

Este proyecto usa una carpeta `.ai/` como fuente de verdad compartida para todas las IAs del equipo.

**Antes de generar código, leer los siguientes archivos en orden:**

1. `.ai/RULES.md` — Reglas de comportamiento
2. `.ai/MODULE_MAP.md` — Mapa de módulos y responsables
3. `.ai/ARCHITECTURE.md` — Arquitectura del sistema
4. `.ai/FINAL_DELIVERY.md` — Checklist de la entrega final

**Skills disponibles en `.ai/skills/`:**
- `analyze-impact.md` — Cómo evaluar el impacto de un cambio
- `clean-architecture.md` — Cómo mantener las capas al escribir código
- `test-pattern.md` — Cómo escribir tests consistentes

**Stack:** Node 18+ · NestJS 10 · TypeORM 0.3 · PostgreSQL 16 · Redis 7 · BullMQ 5 · Docker
